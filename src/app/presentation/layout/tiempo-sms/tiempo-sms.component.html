Tiempo de demora entre que el “Usuario monitor declara la emergencia” y la APP envía los SMS a los contactos de emergencia del usuario final. Es el tiempo que el sistema brinda al usuario monitor cada vez que cierra una alerta para ponerse en condición de pausa. Tiempo que brinda el sistema al usuario monitor para que acepte una nueva alerta asignada. Es el tiempo en el que el sistema reasigna la alerta de un usuario monitor a otro en caso de falla técnica.
